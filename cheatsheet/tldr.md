# tldr

Simplified, community-driven man pages with practical examples.

---

## Usage

| Command | Action |
|---------|--------|
| `tldr command` | Show cheatsheet for command |
| `tldr -u` / `tldr --update` | Update local cache |
| `tldr -l` / `tldr --list` | List all available pages |
| `tldr -p PLATFORM command` | Show for specific platform |
| `tldr -L LANG command` | Show in specific language |

## Platforms

`linux`, `osx`, `windows`, `sunos`, `android`, `common`

```bash
tldr -p linux systemctl
tldr -p osx pbcopy
```
